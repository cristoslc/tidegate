# NousResearch/hermes-agent

**Stars:** 98.7k | **Forks:** 13.9k | **License:** MIT | **Language:** Python 87.5%, TypeScript 8.2%

**Latest release:** v0.10.0 (2026.4.16) | **Commits:** 4,776

## Description

The self-improving AI agent built by Nous Research. Creates skills from experience, improves them during use, and runs anywhere.

## Key Features

| Feature | Description |
|---------|-------------|
| Real terminal interface | Full TUI with multiline editing, slash-command autocomplete, conversation history, interrupt-and-redirect, streaming tool output |
| Lives where you do | Telegram, Discord, Slack, WhatsApp, Signal, and CLI from a single gateway process |
| Closed learning loop | Agent-curated memory with periodic nudges. Autonomous skill creation. FTS5 session search. Honcho dialectic user modeling. agentskills.io compatible |
| Scheduled automations | Built-in cron scheduler with delivery to any platform |
| Delegates and parallelizes | Spawn isolated subagents for parallel workstreams |
| Runs anywhere | Six terminal backends: local, Docker, SSH, Daytona, Singularity, Modal |
| Research-ready | Batch trajectory generation, Atropos RL environments, trajectory compression |

## Supported Models

Nous Portal, OpenRouter (200+ models), NVIDIA NIM, Xiaomi MiMo, z.ai/GLM, Kimi/Moonshot, MiniMax, Hugging Face, OpenAI, custom endpoints.
